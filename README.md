# Omneva

**Native Cross-Platform Media Suite** (Omneva)

Omneva is a powerful desktop application for playing, converting, and transcoding media files. Built with Python and Qt (PySide6) for Windows, macOS, and Linux.

## Features
- **Dual-Engine Player**: Native VLC playback engine for wide compatibility.
- **Transcoder**: Convert video and audio to various formats using FFmpeg.
- **Library**: Manage your local media files.
- **Cross-Platform**: Runs natively on all major operating systems.
- **Theme Support**: Dark and Light themes.

## Usage
Run the executable `Omneva.exe` (Windows) or the binary for your platform.

## Building from Source
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the application:
   ```bash
   python main.py
   ```
3. Build executable:
   ```bash
   python build.py
   ```
